# deku-claude-plugins

A collection of Claude Code plugins by [dev-yakuza](https://github.com/dev-yakuza).

## Installation

```bash
claude /plugin marketplace add dev-yakuza/deku-claude-plugins
claude /plugin install deku-claude-plugins@<plugin-name>
```

## Plugins

| Plugin | Description | Docs |
|--------|-------------|------|
| [sdd-plugin](./plugins/sdd-plugin) | Spec-Driven Development - AI collaborative development process with GitHub integration | [docs](./docs/sdd-plugin.md) |
